# iOS-Interview-RTC

iOS-Interview-RTC 是iOS端Obective-C版本SDK。iOS-SDK 可以帮助您在十分钟内快速搭建基于浏览器的音视频应用。iOS-Interview-RTC 使用WebRTC协议构建音视频通话服务，让您轻松应对一对一、多对多音视频通话及实时数据传输等复杂应用场景。
